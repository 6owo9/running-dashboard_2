# Codex 전용 안내

이 디렉터리는 Codex가 빠르게 참고할 수 있는 축약 규칙을 담습니다. 자세한 규칙은 루트 `AGENTS.md`와 `.claude/rules/`를 함께 확인합니다.

## 적용 범위

- 이 디렉터리의 문서는 Codex 작업 기준입니다.
- `.claude/settings.json`은 Claude Code 권한/훅 설정이며 Codex 런타임에 자동 적용되지 않습니다.
- `.claude/hooks/*.sh`도 Codex가 자동 실행하지 않습니다. 필요한 경우 참고만 합니다.
- `.claude/commands/*.md`는 명령 설명 문서이며 Codex 명령 시스템이 아닙니다.

## 우선순위

1. 사용자 최신 요청
2. `AGENTS.md`
3. `.codex/rules.md`
4. 관련 `.claude/rules/*.md`
5. 기존 코드 패턴

## 작업 방식

- 먼저 읽고, 그다음 수정합니다.
- 변경 범위를 작게 유지합니다.
- 기존 변경사항은 되돌리지 않습니다.
- 검증 결과를 마지막에 짧게 보고합니다.
- 한국어로 답합니다.
- 문서 작업 중에는 앱 코드 변경사항을 정리하거나 되돌리지 않습니다.
